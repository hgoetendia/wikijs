<!-- SUBTITLE: A quick summary of Pip -->

# Install


```sh
sudo yum install python34-pip.noarch
```

or

```sh
sudo yum install python36-pip.noarch

```

# Upgrade pip


```sh
pip install --upgrade pip
```

# Search with pip
Example, search python-dateutil package.

```sh
pip search python-dateutil
```
